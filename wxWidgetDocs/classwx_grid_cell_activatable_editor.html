Detailed Description

Base class for activatable editors.

Inheriting from this class makes it simpler to implement editors that support only activation, but not in-place editing, as they only need to implement TryActivate(), DoActivate() and Clone() methods, but not all the other pure virtual methods of wxGridCellEditor.
