Column width special values.

Enumerator
wxCOL_WIDTH_DEFAULT 

Special value used for column width meaning unspecified or default.

wxCOL_WIDTH_AUTOSIZE 

Size the column automatically to fit all values.

Note
On macOS, this style is only implemented in the Cocoa build on macOS >= 10.5; it behaves identically to wxCOL_WIDTH_DEFAULT otherwise.
