wxDateTime::Tm Struct Reference

#include <wx/datetime.h>

Detailed Description

Contains broken down date-time representation.

This struct is analogous to standard C struct tm and uses the same, not always immediately obvious, conventions for its members: notably its mon and yday fields count from 0 while mday counts from 1.
